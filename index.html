Prompt Ideas

What can you ask DiagramForge?

Anything you'd describe to a colleague at a whiteboard. Here are some prompts to get you started.

Architecture

Draw a C4 context diagram for a banking app with mobile clients, an API gateway, core banking services, and a PostgreSQL database

Microservices

Create a microservices diagram with an auth service, order service, inventory service, and a message queue between them

Data Flow

Show a data pipeline from S3 ingestion through Spark transformation to a Redshift data warehouse with a BI layer on top

Sequence

Create a sequence diagram showing user login flow with OAuth2 — client, auth server, resource server, and token refresh

Iterate

Add a CDN in front of the load balancer and a read replica for the database — keep the existing layout

Cloud

Draw an AWS architecture for a serverless API — API Gateway, Lambda, DynamoDB, and CloudWatch for monitoring

Pro Tips

Get better results faster

🎯
Be specific about diagram type
Say "C4 container diagram" or "sequence diagram" rather than just "diagram" — the AI produces much better output when it knows the target format.
🔗
Name your components
Use real names: "OrderService", "PostgreSQL", "Kafka topic: order-events". Named components give you more precise follow-up edits.
🔄
Iterate in small steps
One change at a time works better than one massive prompt. Start with the overall structure, then add details layer by layer.
✨
Use Gemini 2.5 Pro
For complex architecture diagrams, Gemini 2.5 Pro produces the most accurate draw.io XML. Get a free API key from Google AI Studio.
📐
Ask for layout changes
"Arrange the services in a left-to-right flow" or "group the database components at the bottom" — layout instructions work well.
💾
Export as needed
Use draw.io's built-in File → Export to save as PNG, SVG, or PDF. Your diagram is a standard draw.io file — compatible with any draw.io viewer.
